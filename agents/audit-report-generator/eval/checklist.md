# Audit Report Quality Checklist

Run this after generating every report. ALL must pass.

## Research Quality
- [ ] Website load speed mentioned with specific number (e.g. "4.2 seconds")
- [ ] Google review count cited for the business
- [ ] At least 3 real competitors named with real review counts
- [ ] Mobile responsiveness assessed
- [ ] Contact form / booking capability checked

## Report Content
- [ ] Every [placeholder] replaced with real data — zero template text remaining
- [ ] Score out of 10 is justified with specific findings
- [ ] At least 3 AI opportunities identified with hours-saved estimates
- [ ] ROI calculation uses correct hourly rate for region
- [ ] Tier recommendation matches the selection criteria (not defaulting to Growth)
- [ ] Upgrade path shown in Section 5
- [ ] All competitor cards have real names and real data

## Email Quality
- [ ] If Response Context provided: email opens by referencing their specific words
- [ ] Email body under 150 words
- [ ] CTA button links to correct report URL
- [ ] Subject line includes business name

## Technical
- [ ] HTML saved to both locations (audit-reports/ and founder-dashboard/)
- [ ] Git push to founder-dashboard completed
- [ ] Google Doc backup created
- [ ] Notion pipeline updated
- [ ] Gmail draft created for prospect
- [ ] Summary email SENT (not drafted) to Olushola
