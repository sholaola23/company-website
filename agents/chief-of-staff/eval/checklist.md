# Chief of Staff — Quality Checklist

Run this checklist after every briefing. All items must PASS.

## Completeness (must cover all 8 dimensions)
- [ ] Pipeline health section included with numbers
- [ ] Revenue/financial section included
- [ ] Client health section with per-client status
- [ ] Marketing/ads performance included
- [ ] Agent fleet status included
- [ ] Website/infrastructure status included
- [ ] Risk register with severity ratings
- [ ] Top 3-5 recommended priorities

## Accuracy
- [ ] All numbers sourced from Notion/Gmail (not estimated)
- [ ] Week-over-week comparisons are correct
- [ ] Health scores calculated using the scoring methodology
- [ ] Traffic lights match the score ranges (70+ green, 50-69 amber, <50 red)
- [ ] No hallucinated data — if data unavailable, say "data not available"

## Actionability
- [ ] Every recommendation is specific (names a lead, agent, or action)
- [ ] Priorities are ordered by urgency/impact
- [ ] Risks include severity score AND recommended action
- [ ] No vague recommendations like "improve pipeline" — must say HOW

## Tone
- [ ] Concise — under 500 words total
- [ ] Numbers-first, not adjectives
- [ ] Honest about problems — no sugar-coating
- [ ] Respectful of Olushola's time (he reads this after a full workday)

## Email
- [ ] Subject line includes health score and traffic light
- [ ] Critical risks flagged in subject if severity 8+
- [ ] Sent to olusholaoladipupo1@gmail.com (not drafted)
- [ ] Logged to Notion Sales Agent Reports
