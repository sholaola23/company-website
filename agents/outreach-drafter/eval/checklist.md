# Outreach Email Quality Checklist

## Before Sending
- [ ] Self-quality score is 6+ (if below 6, skip and flag)
- [ ] No existing sent email to this address in Gmail (avoid double-sending)
- [ ] Lead has email address (Needs Email = false)
- [ ] Lead Score >= 50 (or >= 40 fallback)

## Email Content
- [ ] Opens with something about THEIR business, not about us
- [ ] E'Manuel case study adapted for their industry (not "a bakery in Kettering")
- [ ] Uses "AI" as a positive hook (AI-powered, AI assistant, AI booking)
- [ ] No technical jargon (no n8n, webhook, API, automation workflow)
- [ ] Free AI Audit is the primary CTA
- [ ] Tone matches their region (UK/US/Nigeria)
- [ ] Under 6 paragraphs
- [ ] Opt-out line included at the end
- [ ] No two emails in same batch start with same opener

## After Sending
- [ ] Notion status updated to "sent"
- [ ] Sent Date set to today
- [ ] Outreach Date set to today
- [ ] Template Used recorded
- [ ] Last Agent Run set to today
- [ ] Daily report logged
