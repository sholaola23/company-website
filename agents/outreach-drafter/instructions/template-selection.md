# Template Selection Rules

Choose the template based on the lead's pain points and online presence:

| Condition | Template |
|---|---|
| Has website but slow/no response | Template 1: Tested Response Time |
| No website, Facebook-only or social-only | Template 2: No Website |
| Has website but outdated/basic/no booking | Template 3: Website Could Work Harder |
| Lead Score >= 80 (top prospect) | Template 4: Free Demo |

## Decision Priority
1. If Score >= 80 → Template 4 (always for top prospects)
2. If no website → Template 2
3. If website exists but underperforming → Template 1 or 3 (choose based on whether response time or website features are the bigger gap)
4. Default → Template 3

## Subject Line Rules
- Use curiosity and specificity
- Avoid generic openers
- Personalise with business name and location
- See `templates/email-templates.md` for subject line options per template
