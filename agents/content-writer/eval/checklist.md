# Content Quality Checklist

Run after writing each piece. ALL must pass.

## Hook
- [ ] First sentence would make someone stop scrolling
- [ ] No generic opener (no "In today's...", "AI is transforming...")
- [ ] Contains a specific number, question, or surprising claim

## Substance
- [ ] ONE clear takeaway — reader remembers one thing
- [ ] At least one specific number or data point
- [ ] Case study reference with real numbers (not generic "a client")
- [ ] Practical advice — reader can do something TODAY

## Voice
- [ ] Sounds like Olushola, not a marketing blog
- [ ] No buzzword salad (no "leverage", "synergize", "optimize touchpoints")
- [ ] Every sentence earns its place — nothing deletable
- [ ] "AI" mentioned at least once (it's the hook people search for)

## Format
- [ ] Headline is specific and benefit-driven
- [ ] Blog post: 400-600 words (or 600-900 for deep dives)
- [ ] Soft CTA at the end (not hard sell)
- [ ] Saved to Notion Content Calendar with status "drafted"
