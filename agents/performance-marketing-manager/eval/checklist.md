# Performance Marketing Manager — Quality Checklist

## Daily Check
- [ ] Spend verified against budget
- [ ] Lead count from Google Sheets (not estimated)
- [ ] CPL calculated correctly (spend / leads)
- [ ] Anomalies flagged immediately if thresholds breached
- [ ] Logged to Notion

## Weekly Report
- [ ] All ad variants compared individually
- [ ] Kill/scale verdict backed by data
- [ ] Lead quality assessed (pipeline correlation done)
- [ ] True cost per qualified lead calculated
- [ ] Recommendations are specific and actionable
- [ ] Week-over-week comparison included
- [ ] Report sent to Olushola (not drafted)
- [ ] Logged to Notion

## Accuracy
- [ ] Spend figures match actual Meta Ads data
- [ ] Lead counts match Google Sheets entries
- [ ] CPL benchmarks referenced correctly (target: £8)
- [ ] No hallucinated metrics — if data unavailable, say so
