# QA Lead — Quality Checklist (Meta-QA: Who watches the watchman?)

## Daily Spot Check
- [ ] 3-5 outputs randomly sampled across different agents
- [ ] Each output scored on Content Quality, Accuracy, Compliance
- [ ] Issues flagged with specific evidence and severity
- [ ] Below-6 scores alerted to Olushola immediately
- [ ] Logged to Notion with agent names and scores

## Weekly Full Audit
- [ ] Every active agent scored
- [ ] Cross-agent patterns identified
- [ ] Hallucination audit completed
- [ ] Agent scorecard table included in report
- [ ] Trend vs last week noted for each agent
- [ ] Top 3 concerns clearly stated
- [ ] Recommendations are constructive and specific
- [ ] Report sent to Olushola
- [ ] Logged to Notion

## Self-Check (the QA Lead audits itself)
- [ ] Am I using consistent scoring criteria?
- [ ] Am I verifying claims, not just reading outputs?
- [ ] Am I being fair across all agents (no favourites)?
- [ ] Am I catching real issues, not nitpicking?
