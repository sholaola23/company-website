## STEP 0: Read Competitive Intelligence

Before repurposing, read `agents/_shared/competitive-intel.md` for:
- Our key differentiators (instant audit, flat pricing, 7-day delivery, 90-day guarantee)
- Facebook/social post hooks — use these as inspiration for social versions
- Content angles that convert (cost of inaction, before/after, industry-specific)
- Competitor weaknesses to contrast against (generic audits, manual delivery, USD pricing)

Weave these differentiators naturally into repurposed content — especially video scripts and social hooks.

## STEP 1: Read Drafted Content

Query the Content Calendar for pages where:
- Status = "drafted"
- Week = this week

Read the full draft for each piece. Focus especially on the **long post** (blog/Medium) — that's the primary source for repurposing.

If no drafted content found, log "No drafts available — Content Writer may not have run" and stop.

## STEP 2: Repurpose the Long Post

Take the long post and create these versions:

### 2a. Additional Blog Post Angle
Take the main blog post and create a second shorter blog post (300-400 words) approaching the same topic from a different angle:
- Different industry example
- Different pain point
- "Myth vs reality" or "What most people get wrong" framing
This builds SEO depth on the topic with internal linking between related posts.

### 2b. Email Newsletter Snippet (100-150 words)
A short paragraph for a future newsletter:
- Open with the key insight
- Include one specific number or example
- End with: "Want help setting this up? Reply to this email."

### 2c. Short-Form Video Script (60 seconds)
Format:
```
[HOOK — first 3 seconds]
"[Bold statement or question]"

[PROBLEM — 10 seconds]
"[Pain point in 2 sentences]"

[SOLUTION — 20 seconds]
"[What AI can do — specific, not vague]"

[PROOF — 15 seconds]
"[Case study with real numbers]"

[CTA — 10 seconds]
"[What to do next]"
```

Rules: Written for SPEAKING. Short sentences. Conversational. Hook must work with sound off.

## STEP 3: Repurpose the Short Posts

For each short post, create:
- **Alternative hook version** — same content, different opening line
- **Question version** — reframe as a question to drive engagement

## STEP 4: Save All Versions to Notion

Update original pages: add repurposed versions to the body, set Status to "repurposed".

Create NEW pages for each standalone piece:
- Alt Blog Post → Format: "Blog Post", Platform: "Blog", Status: "ready"
- Email Snippet → Format: "Email Snippet", Platform: "Email", Status: "ready"
- Video Script → Format: "Video Script", Platform: "Multi", Status: "ready"

## STEP 5: Log Report

Create a page in Sales Agent Reports:

- Output Summary: "Repurposed [X] pieces into [Y] versions: [X] tweet threads, [X] email snippets, [X] video scripts."

## RULES
- Each platform version must feel NATIVE to that platform. Never just copy-paste.
- Video scripts are highest priority — video gets 10x reach.
- Always include a CTA in every version, but vary the ask.
- Tweets are punchy. Blogs are detailed. Videos are conversational. Emails are personal.
