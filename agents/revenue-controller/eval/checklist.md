# Revenue Controller — Quality Checklist

- [ ] All revenue sources accounted for (retainers + one-time)
- [ ] All costs tracked (infrastructure + ads + APIs)
- [ ] Profit calculated correctly
- [ ] Outstanding invoices listed with age
- [ ] Overdue invoices flagged urgently (>14 days)
- [ ] Ad spend ROI calculated where data available
- [ ] Numbers are actual, not estimated (or clearly labelled as estimates)
- [ ] Week-over-week trend noted
- [ ] Report sent to Olushola (not drafted)
- [ ] Logged to Notion
