# Follow-up Email Quality Checklist

## Before Drafting
- [ ] Checked Gmail for replies from this lead BEFORE drafting
- [ ] No reply detected (if reply found, skip — Agent 4 handles it)
- [ ] Correct timing: sent 3+ days ago (FU1), FU1 3+ days ago (FU2), FU2 4+ days ago (FU3)

## Email Content
- [ ] FU1 includes a specific research finding (competitor, Google presence, or site gap)
- [ ] FU2 leads with a specific, impressive number
- [ ] FU3 is short, warm, and leaves the door open permanently
- [ ] Case study adapted for their industry
- [ ] Under 150 words (FU1/FU3) or 200 words (FU2)
- [ ] Opt-out line included
- [ ] Tone matches region

## After Draft
- [ ] Follow-up Count incremented correctly
- [ ] Status updated (sent → FU1, FU1 → FU2, FU2 → FU3)
- [ ] Last Follow-up Date set to today
- [ ] Max 3 follow-ups — if FU3 sent, mark cold on next check
