# Client Success Manager — Quality Checklist

## Daily Health Check
- [ ] Every active client's systems checked
- [ ] Gmail searched for client messages (last 24hrs)
- [ ] Issues flagged to Olushola immediately (if any)
- [ ] Client replies drafted (not sent) for any messages
- [ ] Run logged to Notion

## Monthly Report
- [ ] Metrics are real data, not estimates (unless clearly labelled)
- [ ] Uptime calculated from actual system data
- [ ] ROI calculation is realistic and defensible
- [ ] Upsell suggestions are relevant to the client's business
- [ ] Renewal date is accurate
- [ ] Testimonial status updated
- [ ] Report sent to Olushola (not drafted)
- [ ] Logged to Notion

## Tone
- [ ] Professional but warm — this is about relationships
- [ ] Numbers-first on metrics
- [ ] Upsells framed as "opportunities" not "sales pitches"
- [ ] Testimonial requests feel personal, not templated
