## 1. Campaign Structure

### Recommended Objective: Lead Generation (Leads)

Use the **Leads** objective, NOT Traffic. Here is why:

- Traffic campaigns optimise for clicks, which attracts window-shoppers
- Lead Generation optimises for people who actually take action (submit the audit form)
- At micro budgets, every pound matters — you want conversions, not vanity clicks
- Meta's AI needs a clear signal of what "success" looks like; form submissions give it that signal

**However:** At £50-100 total spend, you will not hit the ideal 50 conversions/week that Meta's algorithm wants. To work around this:
- Use the **Traffic** objective initially if you get fewer than 5 conversions in the first 3 days
- Switch to Leads once you have enough conversion data for Meta to optimise against
- Alternative: Use Lead Generation with an **Instant Form** (in-app form) to maximise conversions without needing the pixel to fire

### Recommended Setup: Instant Form + Website Redirect

Use Meta's built-in Instant Form to capture name, email, and business type directly inside Facebook/Instagram. After submission, redirect to your audit page so they can also run the instant AI tool. This gives you:
1. A guaranteed lead capture (even if they bounce from the site)
2. A second touchpoint at the audit page
3. Better algorithm optimisation (more conversion signals)

### Budget Allocation

| Scenario | Total Budget | Duration | Daily Budget | Ad Sets | Ads per Set |
|----------|-------------|----------|--------------|---------|-------------|
| **£50**  | £50         | 7 days   | ~£7/day      | 1       | 3 variants  |
| **£100** | £100        | 10-14 days | ~£7-10/day | 1-2     | 3 variants  |

**Key rule:** Never split a micro budget across multiple ad sets. One ad set, three ad variants. Let Meta's AI pick the winner.

For the £100 scenario:
- Days 1-5: Run all 3 ads at £7/day (£35 total) — this is the testing phase
- Days 6-14: Kill the worst performer, reallocate budget to the top 1-2 ads at £7-10/day (£65 remaining)

### Campaign Budget Optimisation (CBO)

Enable **Advantage Campaign Budget** (formerly CBO). This lets Meta auto-distribute spend to the best-performing ad variant. Essential at micro budgets.

### Exclusions

**Not recommended at this budget.** Lookalikes require a source audience of at least 100 people (ideally 1,000+). You do not have enough pixel data yet.

**Build towards it:**
1. Run these campaigns to collect 100+ audit submissions
2. Create a Custom Audience from those submissions
3. Build a 1% Lookalike from that Custom Audience
4. Use the Lookalike in future campaigns

Once your Pixel is installed and collecting data:
- **Website visitors** (last 30 days) who did NOT complete the audit
- **Instant Form openers** who did NOT submit
- **Instagram/Facebook page engagers** (last 90 days)

These retargeting audiences are free to build and extremely effective, but you need traffic first.

### Secondary Test: Worldwide English-Speaking

If the UK audience exhausts quickly (you will see frequency > 2.0), test:
- **Locations:** UK + Ireland + Australia + Canada + USA
- **Same interests and demographics**
- **Separate ad set** (only in the £100 scenario)
- CPC will likely be lower outside UK, but lead quality may vary

- **Keep text on images under 20%** of total area
- **Mobile-first** — 98% of Meta users are on mobile

## 6. Tracking Setup

### 6a. Meta Pixel Installation (Next.js)

Your site is built with Next.js. Here is how to install the Meta Pixel:

**Step 1: Get your Pixel ID**
1. Go to https://business.facebook.com/events_manager
2. Click "Connect Data Sources" > "Web" > "Meta Pixel"
3. Name it "WorkCrew Pixel"
4. Copy the Pixel ID (a number like `123456789012345`)

**Step 2: Create the Pixel component**

Create a new file at `components/meta-pixel.tsx`:

```tsx
'use client'

import Script from 'next/script'
import { usePathname } from 'next/navigation'
import { useEffect } from 'react'

const META_PIXEL_ID = process.env.NEXT_PUBLIC_META_PIXEL_ID

declare global {
  interface Window {
    fbq: (...args: unknown[]) => void
    _fbq: (...args: unknown[]) => void
  }
}

export function MetaPixel() {
  const pathname = usePathname()

  useEffect(() => {
    if (typeof window.fbq === 'function') {
      window.fbq('track', 'PageView')
    }
  }, [pathname])

  if (!META_PIXEL_ID) return null

  return (
    <>
      <Script id="meta-pixel" strategy="afterInteractive">
        {`
          !function(f,b,e,v,n,t,s)
          {if(f.fbq)return;n=f.fbq=function(){n.callMethod?
          n.callMethod.apply(n,arguments):n.queue.push(arguments)};
          if(!f._fbq)f._fbq=n;n.push=n;n.loaded=!0;n.version='2.0';
          n.queue=[];t=b.createElement(e);t.async=!0;
          t.src=v;s=b.getElementsByTagName(e)[0];
          s.parentNode.insertBefore(t,s)}(window, document,'script',
          'https://connect.facebook.net/en_US/fbevents.js');
          fbq('init', '${META_PIXEL_ID}');
          fbq('track', 'PageView');
        `}
      </Script>
      <noscript>
        <img
          height="1"
          width="1"
          style={{ display: 'none' }}
          src={`https://www.facebook.com/tr?id=${META_PIXEL_ID}&ev=PageView&noscript=1`}
          alt=""
        />
      </noscript>
    </>
  )
}
```

**Step 3: Add to your root layout**

In `app/layout.tsx`, import and render the component:

```tsx
import { MetaPixel } from '@/components/meta-pixel'

// Inside the <body> tag:
<MetaPixel />
```

**Step 4: Add the env variable**

In `.env.local`:
```
NEXT_PUBLIC_META_PIXEL_ID=your_pixel_id_here
```

**Step 5: Track audit form submissions**

Wherever your audit form submission happens, fire the Lead event:

```tsx
// After successful form submission:
if (typeof window.fbq === 'function') {
  window.fbq('track', 'Lead', {
    content_name: 'AI Audit',
    content_category: 'audit',
  })
}
```

### 6b. Conversion Events to Track

| Event | When to Fire | Meta Event Name |
|-------|-------------|-----------------|
| Page View | Every page load | `PageView` (automatic) |
| Audit Page View | Visitor lands on /audit | `ViewContent` |
| Audit Started | User begins interacting with audit tool | `InitiateCheckout` |
| **Audit Completed** | User submits audit form | **`Lead`** (primary conversion) |
| Contact Request | User requests a call/meeting | `Schedule` |

The **Lead** event on audit completion is your primary conversion event. This is what you optimise your campaign against.

1. Install the **Meta Pixel Helper** Chrome extension
2. Visit your site — the extension icon should show a green checkmark
3. Go to Events Manager > Test Events and enter your website URL
4. Submit a test audit and verify the Lead event fires

## 7. Testing Plan

### A/B Testing Strategy

With 3 ad variants and a micro budget, you are not running a statistically significant A/B test. Instead, you are doing a **rapid creative test** to find a winner:

**Phase 1: Discovery (Days 1-5)**
- Run all 3 ads simultaneously in ONE ad set
- Advantage Campaign Budget ON (Meta auto-distributes spend)
- Do NOT touch anything for 3-5 days — let the algorithm learn
- Budget: £7/day (£35 total for Phase 1 in £50 scenario)

**Phase 2: Optimise (Days 6+)**
- Review performance after 5 days
- Kill the worst performer (see kill criteria below)
- Reallocate remaining budget to top 1-2 ads

### Kill Criteria (When to Turn Off an Ad)

Turn off an ad if ANY of these are true after 3+ days:

| Metric | Kill Threshold |
|--------|---------------|
| CTR | Below 0.8% |
| CPC | Above £1.50 |
| Cost per Lead (if tracking) | Above £10 |
| Relevance Score / Quality Ranking | Below Average |
| Spend without any clicks | £10+ spent, 0 clicks |

### Scale Criteria (When to Increase Budget)

Increase budget by 15-20% every 48 hours if:
- CPC is below £0.60
- CTR is above 1.5%
- Cost per Lead is below £3
- You are converting audit completions to calls at 5%+

**Never increase budget by more than 20% at once** — this resets the learning phase.

### What to Test Next (After Initial Campaign)

Once you have a winning ad and baseline metrics:
1. **Test new hooks** — keep the winning structure, change the opening line
2. **Test video vs static** — create a 15-second talking-head version of the winner
3. **Test audiences** — duplicate the winning ad into a new ad set with different interests
4. **Test Instant Form vs Website** — see which generates more qualified leads

## 8. Step-by-Step Launch Guide

Follow these steps exactly. Total setup time: 30-45 minutes.

### Prerequisites (Do These First)

- [ ] A personal Facebook account
- [ ] A Facebook Business Page for WorkCrew (create at facebook.com/pages/create if you do not have one)
- [ ] A payment method (debit card, credit card, or PayPal)
- [ ] Ad images/videos created (use Canva — free tier is fine)
- [ ] Meta Pixel installed on your website (see Section 6 above)

### Step 1: Set Up Meta Business Suite

1. Go to **https://business.facebook.com**
2. Click **"Create Account"** (if you do not already have one)
3. Enter business name: **WorkCrew Ltd**
4. Enter your name and business email
5. Click **"Submit"**
6. You will land in Meta Business Suite — this is your hub for everything

### Step 2: Connect Your Facebook Page

1. In Business Suite, click **Settings** (gear icon, bottom left)
2. Click **"Business Assets"** > **"Pages"**
3. Click **"Add Page"**
4. Search for your WorkCrew page and connect it
5. If you do not have a page, click **"Create New Page"** and set one up

### Step 3: Set Up Your Ad Account

1. In Business Suite, click **Settings** > **"Business Assets"** > **"Ad Accounts"**
2. Click **"Add Ad Account"** > **"Create a new ad account"**
3. Name it: **"WorkCrew Ads"**
4. Currency: **GBP (£)**
5. Time zone: **London (GMT)**
6. Click **"Create"**

### Step 4: Add Payment Method

1. Go to **https://business.facebook.com/billing**
2. Click **"Payment Settings"**
3. Click **"Add Payment Method"**
4. Enter your card details or connect PayPal
5. Set a **spending limit** of £100 (protects you from accidental overspend)

### Step 5: Install Meta Pixel

1. Go to **https://business.facebook.com/events_manager**
2. Click **"Connect Data Sources"** (green button)
3. Select **"Web"** > **"Meta Pixel"** > **"Connect"**
4. Name: **"WorkCrew Pixel"**
5. Enter your website URL: **workcrew.io**
6. Choose **"Install code manually"**
7. Copy the Pixel ID
8. Follow the Next.js installation guide in Section 6a above
9. Deploy the site
10. Come back to Events Manager > **"Test Events"** > enter your URL > verify it works

### Step 6: Create Your Campaign

1. Go to **https://adsmanager.facebook.com**
2. Click the green **"+ Create"** button
3. You will see campaign objectives. Select **"Leads"**
4. Campaign name: **"AI Audit - Q1 2026"**
5. Turn ON **"Advantage Campaign Budget"**
6. Set budget: **Daily budget = £7** (or £10 for the £100 scenario)
7. Click **"Next"**

### Step 7: Set Up Your Ad Set

1. Ad set name: **"UK SMB Owners - Interests"**
2. **Conversion location:** Choose **"Instant Forms"** (recommended) OR **"Website"**
3. If Website: select your Pixel and choose **"Lead"** as the conversion event
4. **Budget & Schedule:**
   - Start date: Today (or tomorrow)
   - End date: 7 days from start (£50) or 14 days (£100)
5. **Audience:**
   - Location: **United Kingdom** (type it in, select the country)
   - Age: **25-55**
   - Detailed targeting: Click **"Browse"** > **"Interests"** and add:
     - Small business owners
     - Entrepreneurship
     - Business automation
     - Artificial intelligence
     - Marketing for small business
   - Click **"Exclude People"** and add:
     - University students
     - Enterprise software
6. **Placements:** Keep **"Advantage+ Placements"** selected (default)
7. Click **"Next"**

### Step 8: Create Your Ads

You will create 3 ads within this ad set:

**For each ad:**
1. Ad name: **"Ad 1 - Problem Focused"** (or Ad 2, Ad 3)
2. Identity: Select your WorkCrew Facebook Page
3. Connect your Instagram account (if you have one)
4. **Ad setup:** Select **"Single image or video"**
5. **Media:** Upload your image (1080x1080px for feed, 1080x1920px for stories)
6. **Primary text:** Paste the ad copy from Section 3 above
7. **Headline:** Paste the headline from Section 3

1. Click **"Publish"** (green button, bottom right)
2. Meta will review your ads (usually approved within 1-24 hours)
3. You will get a notification when they are live
4. **Do NOT change anything for the first 3-5 days**

### Step 10: Daily Monitoring (5 minutes/day)

1. Open Ads Manager at **https://adsmanager.facebook.com**
2. Check these columns daily:
   - **Results** (number of leads)
   - **Cost per result** (cost per lead)

1. **Within 5 minutes** of receiving a lead, send a personal email:

   >
   > No pressure at all — just want to make sure you get value from the audit.

   > Best,
   > Olushola
